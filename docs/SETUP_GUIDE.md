# Setup Guide

## Prerequisites
- Node.js 18+
- MongoDB 5+
- Redis 6+

## Installation

1. Clone the repository
2. Copy `.env.example` to `.env`
3. Update `.env` with your configurations
4. Run `npm install` in both backend and frontend
5. Run `npm run dev` in root directory

## Database Setup

Run MongoDB locally or use MongoDB Atlas.
